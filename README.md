# Proyectos de Desarrollo Web

Este repositorio contiene una colección de proyectos de desarrollo web, cada uno demostrando diferentes habilidades y tecnologías. A continuación, se presenta una breve descripción de cada proyecto:

---

## 1. Web-Ejercicios-JavaScriptt

Este proyecto es una plataforma educativa interactiva diseñada para aprender y practicar JavaScript. Incluye una página principal con una introducción al lenguaje, una sección de ejercicios donde los usuarios pueden escribir y ejecutar código JavaScript directamente en el navegador, y una sección de recursos con videos tutoriales. Además, cuenta con una subsección de 'Apuntes' que ofrece tutoriales detallados sobre JavaScript, CSS y Angular, cubriendo desde conceptos básicos hasta avanzados.

**Tecnologías:** HTML, CSS, JavaScript, Angular (apuntes).

---

## 2. Print-PDF

Este proyecto es una aplicación web sencilla que permite visualizar un currículum vitae (CV) y descargarlo como un archivo PDF. Utiliza HTML para la estructura del CV, CSS para el diseño y JavaScript, junto con las librerías `html2canvas` y `jspdf`, para generar el PDF a partir del contenido HTML de la página.

**Tecnologías:** HTML, CSS, JavaScript, `html2canvas`, `jspdf`.

---

## 3. slider-wallpaper

Este proyecto implementa un carrusel de imágenes (slider) con efectos visuales y navegación interactiva. Las imágenes se cargan dinámicamente desde un array de JavaScript, y el slider incluye miniaturas para una navegación rápida, así como botones para avanzar y retroceder. El diseño está cuidadosamente estilizado con CSS para ofrecer una experiencia de usuario atractiva y animaciones suaves.

**Tecnologías:** HTML, CSS, JavaScript.

---

## 4. Tienda Informatica

Este proyecto es una página web para una tienda de informática llamada 'InforPC'. Incluye una página de inicio que destaca servicios como ordenadores a medida, presupuestos sin compromiso, un mapa de ubicación y ofertas de intercambio. También cuenta con una sección de contacto con un formulario y una página de tienda que actualmente está en construcción. El diseño utiliza HTML y CSS, con algunos efectos de animación implementados con jQuery.

**Tecnologías:** HTML, CSS, JavaScript, jQuery.

---

## 5. Juegos

Este proyecto es una colección de juegos clásicos implementados en JavaScript, HTML y CSS. La página principal actúa como un portal, mostrando miniaturas de cada juego (Ahorcado, Buscaminas, Memory Game, Snake, Tetris, Tres en Raya) que enlazan a sus respectivas implementaciones. Es una demostración de habilidades en desarrollo de juegos web básicos.

**Tecnologías:** HTML, CSS, JavaScript.

---

## 6. Pokemon

Este proyecto es una Pokedex interactiva que permite buscar y visualizar información de Pokémon. La aplicación cuenta con una interfaz de usuario para buscar Pokémon por nombre, navegar entre ellos y reiniciar la búsqueda. Está construida con HTML, CSS y JavaScript, y se espera que interactúe con una API de Pokémon para obtener los datos. Incluye documentación sobre su creación.

**Tecnologías:** HTML, CSS, JavaScript.

---

## 7. Portafolios_SASS

Este proyecto es un portafolio personal desarrollado con HTML, SASS (compilado a CSS) y JavaScript. Presenta las habilidades, proyectos y experiencia laboral del desarrollador en un formato atractivo y responsivo. Destaca el uso de SASS para una gestión de estilos modular y eficiente, incluyendo un carrusel interactivo para mostrar proyectos y una navegación fluida entre secciones.

**Tecnologías:** HTML, SASS, CSS, JavaScript.
